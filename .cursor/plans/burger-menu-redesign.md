# Редизайн burger-меню LOCUS

## Цель
Белое slide-меню слева уровня приложения (как на референсе). Только фронт, логика не меняется.

## Статус: выполнен

## Сделано (актуальная версия)

### Контейнер и overlay
- Drawer **слева**: `left: 0`, `translateX(-100%) → 0`, `88%` / `max-width: 380px`, `background: #ffffff`, высота `100vh`, скругления справа `24px`, тень, анимация `240ms cubic-bezier(0.22, 1, 0.36, 1)`.
- Overlay: `rgba(0,0,0,0.25)`, `backdrop-blur: 3px`, opacity 0→1.

### Кнопка закрытия
- Одна кнопка: иконка `ArrowLeft` (назад) 28px, `top: 18px; right: 18px`, цвет `#6f6f6f`, hover `#000`. Крестика нет.

### Шапка меню (профиль)
- Контейнер: `padding: 20px 20px 12px`, клик → `/profile`. Аватар 44px, градиент `#7b61ff` → `#9f7bff`, буква имени (или «Г» для гостя). Имя 16px 600 `#111`, подпись «Перейти в профиль» 13px `#8a8f98`, margin-top 2px.

### Основные пункты
- Контейнер: `margin-top: 18px`, `padding: 0 10px`. Пункты: высота 48px, `border-radius: 12px`, `padding: 0 12px`, `gap: 12px`, hover `#f5f6fa`. Иконки 20×20, `#8f96a3`, при hover `#7b61ff`. Список: Поиск жилья, Избранное, Сообщения, Профиль, Тарифы, Помощь.

### Кнопка «Войти / Зарегистрироваться»
- Показывается только когда не авторизован. Обёртка `.mobile-menu-cta-wrap`, кнопка градиент, 44px, 12px radius.

### Нижний блок
- «Выйти» только для авторизованных: `margin-top: auto`, цвет `#ff4d4f`, иконка выхода слева, padding 14px 10px 24px.

### Файлы
- `frontend/styles/globals.css` — стили drawer (слева, белый), overlay, profile block, close, menu list, logout.
- `frontend/shared/ui/HeaderLight.tsx` — ArrowLeft, профиль с `user?.full_name` / «Гость», список пунктов, Выйти внизу. Без соцсетей, без дубля логотипа.

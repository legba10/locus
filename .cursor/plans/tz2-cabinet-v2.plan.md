# ТЗ 2 — Полный редизайн кабинета пользователя (Owner/Host Dashboard v2)

## Цель
Убрать пустоту и перегруженность, дубли кнопок «Разместить», добавить логику и структуру уровня Airbnb/Cian. Понятно за 3 секунды, масштабируемо, без поломки backend.

## Файлы

- `frontend/config/uiFlags.ts` — useCabinetV2
- `frontend/app/owner/dashboard/DashboardV2.tsx` — кабинет v2 (главная, вкладки, без дублей кнопки)
- `frontend/app/owner/dashboard/SidebarV2.tsx` — сайдбар v2 (иконки, hover, active, mobile)
- `frontend/components/cabinet/ListingCardCabinetV2.tsx` — карточка объявления в кабинете (фото, цена, статус, просмотры, избранное, редактировать/скрыть/удалить/поднять/архив)
- `frontend/components/cabinet/index.ts` — экспорт
- `frontend/app/owner/dashboard/PageClient.tsx` — переключение по флагу (useCabinetV2 ? DashboardV2 : OwnerDashboardV7)

## Переключение
При `useCabinetV2 === true` рендерится `DashboardV2`, иначе `OwnerDashboardV7`. Кнопка «Разместить» только в header сайта, внутри кабинета дубли не рендерятся.

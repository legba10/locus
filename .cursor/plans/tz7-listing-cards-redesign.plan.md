# ТЗ 7 — Редизайн карточек объявлений (уровень Cian/Airbnb)

## Цель
Информативная, плотная, премиальная карточка. Ответ за 2 секунды: что, где, сколько, почему, можно ли сегодня. Не ломать API/данные.

## Структура карточки
1. **Фото** — высота ПК 220px, моб 200px; 4/3 cover. Оверлеи: слева сверху бейджи (Проверено, AI рекомендует, Топ вариант), справа ♡ избранное, снизу градиент + тип + город • район.
2. **Основной блок** — цена крупно + «или X / месяц»; характеристики в строку (гости, комнаты, м², метро); AI-метка; даты (Свободно сегодня / С даты).
3. **Доверие** — просмотры за неделю, сохранения.
4. **Кнопка** — Смотреть / Забронировать.
5. **Стиль** — padding 12px, border-radius 16px, gap 8px; hover: подъём, тень, затемнение фото.
6. **Сетка** — ПК 3–4 в ряд, моб 1; отступы моб 16px.

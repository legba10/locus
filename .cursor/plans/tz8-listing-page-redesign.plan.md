# ТЗ 8 — Редизайн страницы объявления (детальная)

## Цель
Информативная, доверительная страница: понятно за 5 секунд, цена и даты, бронирование без путаницы. Только UI.

## Структура сверху вниз
1. Галерея — ПК: большое слева + 4 мини справа, 420px; моб: слайдер, счётчик 1/12; клик → fullscreen
2. Основной блок — заголовок, подзаголовок, цена крупно + /месяц, метрики (гости, комнаты, м², метро)
3. Блок бронирования — справа ПК / снизу моб, фиксированная карточка: даты, гости, расчёт, кнопка
4. Исправить баг дат — одинаковая высота/ширина, border-radius 12px
5. AI-блок — «Подобрано для вас», кнопка «Почему?» → tooltip
6. Описание — «Показать полностью»
7. Удобства — сетка с иконками
8. Хозяин — аватар, имя, рейтинг, «Написать»
9. Район — карта, метро, время
10. Похожие — слайдер внизу
11. Моб — порядок: фото, цена, даты, кнопка, описание; кнопка брони фиксирована снизу

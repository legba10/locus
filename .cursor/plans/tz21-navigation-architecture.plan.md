# ТЗ-21 — Финальная архитектура навигации

Не удалять функционал. Только перенос, объединение, упорядочивание.

## Уровни навигации

- **A — Шапка:** быстрые действия (логотип, поиск, сообщения, уведомления, аватар)
- **B — Нижнее меню (mobile):** основные разделы (Главная, Поиск, Добавить, Сообщения, Профиль)
- **C — Меню аватара:** личный кабинет (Профиль, Мои объявления, Бронирования, Доход, Настройки, Выйти; админ — Админ панель)

## Где что оставить

| Элемент       | Где оставить           |
|---------------|------------------------|
| Сообщения     | шапка + нижнее меню    |
| Профиль       | меню аватара (+ нижнее меню как вход) |
| Разместить    | нижнее меню            |
| Избранное     | только в поиске/профиле (не в шапке и не в низах) |

## Выполнено

- Шапка: логотип, поиск, сообщения, уведомления, аватар (без избранного)
- Меню аватара: Профиль, Мои объявления, Бронирования, Доход, Настройки, Выйти; один пункт «Админ панель» для админа
- Нижнее меню: Главная, Поиск, Добавить объявление, Сообщения, Профиль
- Настройки ведут на /profile/settings

# ТЗ-46.3 — Мастер объявления, карты, фото, AI описание, модерация

## Цель
Стабилизировать создание/редактирование объявления: единый модуль формы, корректные шаги, карта и адрес, типы фото, AI-описание с подтверждением, и логика повторной модерации только для чувствительных изменений.

- [completed] Перевести страницы создания/черновика на новый `listingWizard`.
- [completed] Обновить UI шага адреса: контраст города, автодополнение, подсветка улицы, карта/пин.
- [completed] Реализовать типы фото и действия (тип, обложка, порядок, удаление) в новом модуле.
- [completed] Добавить AI-генерацию описания с подтверждением перед вставкой в `textarea`.
- [completed] Реализовать backend-логику модерации при редактировании: pending только при изменении фото/адреса/типа.

# ТЗ-13 — Карточки объявлений (главная / поиск / категории)

## Цель
Карточки уровня Airbnb/Cian с AI-метриками. Решение по карточке без перехода внутрь.

## Структура карточки
1. Фото (4:3, max height) + overlay: слева бейджи (Проверено, AI-подбор, Новое), справа ♡
2. Цена + рейтинг (3 500₽ / ночь ★ 4.9), ниже «от N ночей» или «Новое объявление»
3. Локация: город · район, метро
4. AI-метрики: «Подходит для: …» или «AI: 92% совпадение» + tooltip
5. Параметры: 1к · 42м² · 4 гостя
6. Кнопки: [♡] [Посмотреть]

## Состояния
- Нет фото → плейсхолдер
- Нет цены → не показывать цену
- Новый объект → бейдж NEW

## Сетка
- Desktop: 3 колонки, gap 24px
- Mobile: 1 колонка, gap 16px

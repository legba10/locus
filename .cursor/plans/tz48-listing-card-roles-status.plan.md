# ТЗ-48 — Полная стабилизация карточки объявления (роли + статусы)

## Цель
Единая архитектура карточки объявления: корректные статусы, фиксированная нижняя панель, чёткие действия по ролям (пользователь / арендодатель / админ).

## Шаги
- [completed] Строгая система статусов: draft→Черновик, pending→На модерации, active→Активно, rejected→Отклонено, hidden→Скрыто. Никогда не показывать «На модерации» при status === active.
- [completed] Нижняя панель: position fixed, z-50, backdrop-blur 20px, padding-bottom env(safe-area-inset-bottom).
- [completed] Карточка арендодателя: Редактировать, Календарь, Продвижение, Аналитика, Скрыть, Удалить (все на странице).
- [completed] Карточка админа: Редактировать, Модерация, Жалобы, Статистика, Скрыть, Удалить; статус только через getListingStatusLabel.
- [completed] Блок цены/действий: вертикальный стек (цена, адрес, рейтинг, кнопки).
- [completed] Галерея mobile: высота 280px.
- [completed] Commit и push.

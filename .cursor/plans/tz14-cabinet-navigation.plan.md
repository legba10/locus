# ТЗ-14 — Кабинет и навигация (без дублей)

## Цель
Одна логичная структура: header минимальный, нижнее меню — главный центр, профиль с вкладками.

## Роли
Гость | Пользователь | Арендодатель | Админ (только /admin).

## Изменения
1. **Header (авторизован)**: только [логотип] [поиск] [колокольчик]. Убрать: сообщения, профиль, доходы, разместить.
2. **Нижнее меню**: уже Главная, Поиск, Добавить, Сообщения, Профиль.
3. **Desktop**: левый sidebar с теми же 5 пунктами + Выйти.
4. **Меню аватарки**: не в header; в sidebar только навигация.
5. **Профиль**: вкладки Мои объявления, Бронирования, Финансы (если арендодатель), Настройки. Убрать дубли: избранное из 3 мест (оставить 1), сообщения, доходы из обычного профиля, админ из настроек.
6. **Доходы**: только если есть объявления.
7. **Админ**: только через /admin, убрать из настроек/сайдбара.

# ТЗ-15 — Редизайн кабинета пользователя и арендодателя

## Цель
Кабинет уровня продукта: два режима (пользователь / арендодатель), без дублей.

## Сделано
- **ProfileSidebar**: группы Основные (Мои объявления, Бронирования, Сообщения, Избранное), Для арендодателя (Доходы, Аналитика, Продвижение), Системные (Настройки, Поддержка). Профиль — первый пункт.
- **Profile page**: аватар, имя, город, рейтинг, кнопка «Редактировать профиль». Убраны блоки «Разделы» и «Отзывы» (навигация только в сайдбаре).
- **Аналитика**: страница /profile/analytics с метриками (рейтинг, отзывы, загрузка, конверсия) для арендодателя.
- **Пустое состояние**: «У вас пока нет объявлений» + кнопка [Разместить] во вкладке «Мои объявления».
- **Настройки**: без доходов и дашборда (уже по ТЗ-14).

## Не менять
- Нижнее меню (ТЗ-14).
- Админ только через /admin.

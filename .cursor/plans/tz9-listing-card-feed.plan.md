# ТЗ №9 — Карточка объявления в ленте (главная, список, поиск)

## Цель
Сделать карточку объявления читаемой, премиальной, без засветов, удобной на мобиле, кликабельной. Продуктовый уровень.

## Статус: ✅ выполнено

### Сделано

1. **Структура карточки**
   - [фото] → [цена] → [название] → [район] → [мета].

2. **Блок фото**
   - Высота 220px, radius 18 (сверху), placeholder при отсутствии фото.
   - Свайп по фото (touch + кнопки стрелок на десктопе), индикатор точек.
   - Лайк не добавлен (ТЗ: не добавлять).

3. **Цена**
   - В левом нижнем углу фото: `background: rgba(0,0,0,0.6)`, `color: white`, padding 6px 10px, radius 10, font-weight 600. Формат: «12 000 ₽ / мес» или «Цена по запросу».

4. **Название**
   - Под фото: `color: var(--text-primary)`, font-weight 600, line-clamp 2.

5. **Район**
   - Строка «Москва • Таганская»: `color: var(--text-secondary)`, font-size 14px.

6. **Мета**
   - «2 комнаты • 45 м² • 3 этаж» (или «3 из 9 этаж»): `color: var(--text-muted)`, font-size 13px.

7. **Контейнер карточки**
   - `background: var(--bg-card)`, `border: 1px solid var(--border-main)`, radius 18, padding-bottom 12 (pb-3), overflow hidden.

8. **Сетка**
   - Между карточками gap 12px (gap-3). Мобильная: 1 колонка. Планшет: 2 колонки (sm:grid-cols-2). На странице поиска добавлен lg:grid-cols-3 для десктопа.

9. **Устранение засветов**
   - Все тексты через var(--text-primary), var(--text-secondary), var(--text-muted). Без жёсткого text-white вне оверлея цены (оверлей — по ТЗ rgba+white).

10. **Клик**
    - Вся карточка — ссылка `Link` на `/listings/[id]`. Сохранён трекинг listing_view и locus_viewed_count.

11. **Компонент**
    - Обновлён `components/listing/ListingCard.tsx`: тот же интерфейс props, упрощённый рендер под ТЗ-9. Убраны бейджи, избранное, владелец, AI-строка, кнопка «Смотреть» — только база.

12. **Скелетон**
    - ListingCardSkeleton: блок фото 220px, затем 3 линии (название, район, мета).

13. **Места использования**
    - Главная (HomePageV6), поиск (SearchPageV4), избранное (PageClient), похожие (ListingLayout), профиль пользователя (user/[id]) — везде сетка с gap-3 и grid-cols-1 sm:grid-cols-2 (и lg:grid-cols-3 где было).

### Критерии готовности (ТЗ №9)
- ✔ Карточки читаемы
- ✔ Нет засветов (тема)
- ✔ Одинаковые размеры (фото 220px, единый контейнер)
- ✔ Кликабельны (вся карточка → /listings/id)
- ✔ Mobile ок (1 колонка, свайп)

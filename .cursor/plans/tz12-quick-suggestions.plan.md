# ТЗ-12: Быстрые подсказки сообщений в чате

## Статус: выполнен

## Сделано

### 1. Где показываются подсказки
- Показываются, пока пользователь ещё не отправил ни одного сообщения в этом чате (`!hasUserSent`).
- Скрываются после первой отправки.

### 2. Расположение
- Блок между областью сообщений и полем ввода (под чатом, над инпутом).

### 3. Дизайн
- Карточки-чипы: `rounded-full`, компактные, без эмодзи, стиль сайта (`bg-[var(--bg-secondary)]`, `border-[var(--border)]`).

### 4. Логика
- По нажатию на чип текст вставляется в поле ввода (`setInput(text)`). Пользователь может править и отправить. Автоотправки нет.

### 5. Набор шаблонов (7 базовых)
1. Здравствуйте! Можно забронировать жильё?
2. Свободно ли на эти даты?
3. Можно посмотреть квартиру сегодня?
4. Подскажите, какая окончательная цена?
5. Можно заселиться раньше?
6. Есть ли парковка?
7. Подходит ли для долгого проживания?

### 6. Динамическая подсказка
- При наличии `suggestedCheckIn` и `suggestedCheckOut` (из URL или пропсов) добавляется подсказка: «Здравствуйте! Хочу забронировать на {день месяц}–{день месяц}.»
- Страницы чата и сообщений читают `checkIn`/`checkOut` из query и передают в `ChatPanel`.

### 7. Мобильная версия
- Горизонтальный скролл (`overflow-x-auto`), чипы в одну строку, layout не ломается.

### 8. Тёмная тема
- Стили в `messages-tz11.css`: тёмные чипы `rgba(255,255,255,0.08)`, граница и hover для dark.

### 9. Анимация
- Плавное появление: `fade + slide up` (keyframes `chat-suggestions-fade-slide` в CSS).

## Файлы
- `frontend/components/chat/ChatPanel.tsx` — константа шаблонов, логика показа, блок подсказок, динамическая подсказка по датам.
- `frontend/styles/messages-tz11.css` — стили и анимация для `.chat-suggestions-tz12` и чипов, тёмная тема.
- `frontend/app/chat/[id]/page.tsx` — передача `suggestedCheckIn`/`suggestedCheckOut` из `searchParams`.
- `frontend/app/messages/page.tsx` — передача дат из query в `ChatPanel` (при наличии).

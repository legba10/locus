# ТЗ-16: Финальная полировка UX, анимаций и ощущения продукта

## Цель
Довести интерфейс до ощущения дорогого сервиса: плавность, живость, предсказуемые реакции. Без тяжёлых библиотек — только CSS.

## Глобальная анимационная система

- **Токены:** `--motion-duration: 0.22s`, `--motion-duration-slow: 0.25s`, `--motion-ease: cubic-bezier(0.4, 0, 0.2, 1)`.
- **Учёт reduced-motion:** при `prefers-reduced-motion: reduce` длительности сведены к 0.01s.

## Реализовано

### 1. Карточки объявлений
- Hover (desktop): `translateY(-4px)`, мягкая тень, `cursor: pointer`.
- Transition: `transform` и `box-shadow` с единым ease.

### 2. Кнопки
- Hover: `scale(1.02)`, плавный переход.
- Active: `scale(0.98)`.
- Применено к основным CTA и кнопкам в хедере.

### 3. Иконки в хедере
- Hover: `opacity: 0.85`, `scale(1.05)`.
- Transition на ссылках и кнопках в `.layout-header__right`.

### 4. Бургер / мобильное меню
- Overlay: `backdrop-filter: blur(6px)`, затемнение, переход opacity 0.25s.
- Drawer: transition `transform` 0.25s с `--motion-ease`.

### 5. Появление страницы
- `.main-with-header`: лёгкий fade-in (opacity 0 → 1) при первой загрузке.

### 6. Всплывающие окна / модалки
- `.modal-panel`: анимация появления `scale(0.95) → 1` + fade.
- Выпадающие меню профиля без scale (только opacity), чтобы не ломать вёрстку.

### 7. Фильтры
- Чипы: плавный переход цвета/фона/границы; active `scale(0.97)`.
- При выборе фильтра на поддерживаемых устройствах: `navigator.vibrate(10)` (микро-отдача на mobile).

### 8. Переключатель темы
- Плавный переход фона/цвета у `html`/`body` (0.25s).
- У кнопки темы: transition фона, границы, `transform`; active `scale(0.97)`.

### 9. Сетка карточек (главная / каталог)
- Элементы `.listing-grid > *`: лёгкое появление (fade + небольшой translateY) с небольшим stagger по задержке.

### 10. Bottom Sheet
- Панель и overlay: transition 0.25s для `transform` и `opacity`.

### 11. Уведомления
- Панель уведомлений: появление через slide сверху + fade (keyframes в motion-tz16).

### 12. Mobile
- Tap feedback: у `.listing-card` на touch-устройствах `:active` даёт `scale(0.99)`.
- Вибрация при выборе фильтра (см. выше).

### 13. Скролл
- В `globals.css` уже задано `scroll-behavior: smooth` для `html`.

### 14. Скелетоны
- Используются существующие `ListingCardSkeleton` и скелетоны страниц (в т.ч. профиль/листинг) — без изменений.

## Запреты соблюдены

- Нет тяжёлых анимационных библиотек.
- Нет Lottie в общих компонентах.
- Только лёгкие CSS-анимации и переходы.

## Результат

Интерфейс стал плавнее и предсказуемее: единый motion-стиль, отзывчивые hover/active, мягкое появление блоков и модалок, плавная смена темы и микро-отдача на фильтрах и тапах.

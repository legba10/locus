# ТЗ 1 — Полный редизайн карточки объявления (Listing UI v2)

## Цель
Убрать «пустоту» объявлений: фото 80% экрана, нет информативности, быстрых действий и доверия. Новый формат — информативнее Циан, компактнее, быстрее считывается, продукт 2026. Без изменений backend/API/маршрутов/авторизации.

## Статус

| Часть | Описание | Статус |
|-------|----------|--------|
| 1 | Блоки 1–5 карточки (фото+данные, основная инфа, параметры, описание, быстрые действия) | ✅ |
| 2 | Мини-галерея (карусель, 5 превью, свайп, счётчик) | ✅ |
| 3 | Режим компактных карточек (список, высота ≤220px) | ✅ |
| 4 | Страница объявления: галерея (полноэкран, планировка, видео, 3D), инфо, AI-блок | ✅ |
| 5 | Статусы доверия (проверено, собственник, агент, супер-хозяин) | ✅ |
| 6 | Запрет ломать backend/API/маршруты/авторизацию | ✅ |
| 7 | Компоненты ListingCardV2, ListingPageV2; флаг useNewListingUI | ✅ |
| 8 | Проверка: богато, не пусто, макс. инфа, компактно | ✅ |

## Файлы

- `frontend/config/uiFlags.ts` — `useNewListingUI`
- `frontend/components/listing/ListingCardV2.tsx` — новая карточка
- `frontend/domains/listing/ListingPageV2TZ1.tsx` — новая страница объявления (ТЗ1)
- `frontend/app/listings/[id]/PageClient.tsx` — переключение по флагу
- `frontend/components/listing/index.ts` — экспорт карточки с переключением по флагу

## Переключение
- Карточки: при `useNewListingUI === true` рендерится `ListingCardV2`, иначе текущая `ListingCard`.
- Страница: при `useNewListingUI === true` рендерится `ListingPageV2TZ1`, иначе `ListingPageLight` (V3).

# ТЗ-3: Главная — Hero + поиск + логика входа в подбор

**Цель:** единый сценарий поиска жилья за ≤7 действий; убрать пустоты и нелогичный порядок блоков.

## Статус: ✅ Выполнено

## Что сделано

1. **Порядок блоков (жёстко):**
   - 1. Hero (приветствие, печатная строка, CTA)
   - 2. Умный подбор (AI) — отдельная карточка под Hero
   - 3. Быстрый поиск (#search) — город, бюджет, тип, кнопки
   - 4. Популярные города
   - 5. Актуальные объявления
   - 6. … (рекомендации, просмотренные, новости, статистика)

2. **Кнопка «Подобрать жильё»:**
   - По умолчанию: плавный скролл к блоку `#search` на этой же странице.
   - Если город уже выбран: переход к результатам поиска с индикатором загрузки.

3. **Hero:**
   - Убрана встроенная AI-панель из Hero; AI вынесен в отдельную секцию под Hero.
   - Печатные фразы дополнены: «Проверенные объявления», «За 7 кликов» (полный набор по ТЗ).
   - Отступы Hero уменьшены (pt/pb), без больших пустот.

4. **Отступы:**
   - Между верхними блоками: 32px (mobile), до 40px (desktop); класс `home-tz3-block`.
   - Убран лишний `margin-top` у блока поиска в контексте `.home-tz3`.

5. **Фильтры:**
   - Кнопка «Фильтры» по-прежнему открывает модалку/ BottomSheet, не отдельную страницу.

# ТЗ №6 — Страница «Мои объявления» и список объявлений пользователя

## Цель
Сделать раздел «Мои объявления» читаемым, структурированным, с нормальными статусами, без засветов в dark theme, удобным на мобильном.

## Статус: ✅ выполнено

### Сделано

1. **Структура страницы**
   - Заголовок «Мои объявления», блок с тарифом и счётчиком, список карточек, sticky-кнопка «Разместить объявление» внизу.

2. **Карточка объявления** (`components/listing/MyListingCard.tsx`)
   - Фото обложки 96×96, radius 12, placeholder при отсутствии.
   - Название (font-weight 600, `var(--text-primary)`), цена (`var(--text-primary)`), вторичный текст (`var(--text-secondary)`).
   - Верстка: radius 16, padding 12, `bg-[var(--bg-card)]`, `border border-[var(--border-main)]`.

3. **Статусы** (`components/listing/StatusBadge.tsx`)
   - Активно (зелёный), На проверке (оранж), Скрыто (серый), Архив (muted), Черновик (серый).
   - Маппинг API: PUBLISHED → active, PENDING_REVIEW → pending, ARCHIVED → archived, DRAFT → draft, REJECTED/BLOCKED → hidden.

4. **Действия в карточке**
   - Редактировать, Скрыть, Удалить.
   - На мобильном — меню по нажатию на три точки (то же три действия).
   - Скрыть = `POST /api/listings/:id/unpublish`, Удалить = `DELETE /api/listings/:id` с подтверждением.

5. **Пустое состояние**
   - Текст «У вас пока нет объявлений», кнопка «Разместить объявление», блок центрирован.

6. **Тёмная тема**
   - Все тексты через `var(--text-primary)` / `var(--text-secondary)` / `var(--text-muted)`; карточки через `var(--bg-card)` и `var(--border-main)`.

7. **Sticky-кнопка**
   - Внизу страницы вкладки «Мои объявления»: фиксированная полоса с кнопкой «Разместить объявление».

8. **Маршрут /my-listings**
   - `app/my-listings/page.tsx` — редирект на `/owner/dashboard?tab=listings`.

### Критерии готовности (ТЗ №6)
- ✔ Карточки читаемы
- ✔ Статусы есть
- ✔ Нет засветов (использованы CSS-переменные темы)
- ✔ Пустое состояние есть
- ✔ Кнопка «Разместить» (в шапке при наличии объявлений и sticky внизу)

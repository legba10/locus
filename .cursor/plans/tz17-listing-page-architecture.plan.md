# ТЗ-17: Финальная архитектура страницы объявления

## Цель
Перестроить логику страницы объявления без удаления функционала. Структура уровня Airbnb/Cian: один поток принятия решения.

## Порядок блоков (итог)

1. Фото (галерея)
2. Цена (мобилка — блок под фото; ПК — правая колонка)
3. Липкий бар брони (мобилка — внизу: [Написать] [Забронировать])
4. AI анализ
5. Описание (5 строк + раскрытие)
6. Удобства (сетка + «показать все»)
7. Владелец
8. AI-метрики квартиры (AIMetricsCardTZ9)
9. Отзывы
10. Район + карта

# ТЗ 9 — Блок отзывов + AI-метрики (Trust Layer)

## Цель
Отзывы и AI-метрики как ключевое преимущество (сильнее ЦИАН/Avito/Airbnb). Только UI/UX, без удаления, без поломки бронирования/сообщений/карточки.

## Порядок блоков (Trust Layer)
После описания и удобств:
1. Хозяин + рейтинг
2. AI-метрики квартиры
3. Отзывы
4. Ответы хозяина (в карточке отзыва)
5. CTA бронирования (уже справа/снизу)

## AI-метрики
- Карточка: «AI-оценка квартиры», общий индекс 82/100
- Метрики: Чистота, Безопасность, Район, Цена/качество, Тишина, Транспорт, Оснащение, Хозяин
- Каждая: процент, цвет, tooltip по клику («Оценка на основе: 24 отзывов, 18 бронирований, данных района»)
- Логика расчёта — позже, UI подготовлен

## Блок отзывов
- Верх: «Отзывы гостей», 4.8 ★, «На основе N отзывов», «92% рекомендуют», одна кнопка «Оставить отзыв»
- Распределение: 5★ █ 70%, 4★ █ 15% …
- AI-сводка: «Гости хвалят: чистота, район…» / «Недостатки: …»
- Фильтры: горизонтальный скролл (Все, 5★, С текстом, Последние, Чистота, Район, Цена)
- Карточка отзыва: аватар, имя, дата, оценка, текст, оценки по метрикам, блок «Ответ владельца» (если есть)
- Sticky-панель снизу не ломать (Цена, Написать, Забронировать, Избранное)

## Мобильная
- AI-метрики компактно, отзывы карточками, фильтры свайп, кнопка «Оставить отзыв» одна (sticky или в блоке)

## Анимации
- Метрики — плавная загрузка; рейтинг — счётчик; отзывы — fade-in

# ТЗ 2 — Визуализация отзывов и аналитики

## Статус: выполнен

## Сделано

### Страница объявления
- **Верхний агрегат:** ★ 4.8, 92%, «на основе N отзывов», кнопка «Оставить отзыв» (якорь на форму).
- **Гистограмма:** 5★ … 1★ с процентами и полосками.
- **Блок «Метрики жилья»:** карточка с 5–6 метриками (Чистота, Безопасность, Соответствие, Заселение, Цена/качество, Коммуникация), данные из GET /reviews/listing/:id/metrics.
- **Фильтры:** все | только 5★ | с текстом | последние (фильтрация на фронте).
- **Список отзывов:** карточки с звездой, датой, текстом, мини-метриками, аватар/инициал автора. Пагинация «Показать ещё» (skip).
- **После POST:** инвалидация listing-reviews, listing-rating-summary, listing-metrics; сброс additionalReviews.
- **Skeleton** при загрузке списка отзывов.

### Профиль владельца
- Блок рейтинга показывается **только при просмотре чужого профиля** (id !== currentUser.id).
- Под именем: ★ 4.9, 96%, N отзывов (данные из GET /reviews/user/:userId/summary).
- Свой профиль: по-прежнему «Рейтинг X · N отзывов» без блока аналитики.

### Бэкенд
- GET /reviews/listing/:id поддерживает **skip** для пагинации.
- Ответ списка отзывов включает **author** (id, profile.name, profile.avatarUrl) и **metrics** (metricKey, value).

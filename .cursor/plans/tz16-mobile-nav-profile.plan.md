# ТЗ-16 — Реструктуризация мобильной навигации и профиля

## Цель
Восстановить логичную архитектуру: нижнее меню (основное), шапка (быстрые действия), профиль (кабинет). Админ только через аватар.

## Сделано
- **Шапка (авторизован)**: [логотип LOCUS] [поиск] [колокол] [аватар]. Логотип по центру на mobile (grid 1fr auto 1fr). Бургер только для гостя.
- **Аватар**: при клике — мини-меню: Профиль, Админ панель (если admin), Выйти. Закрытие по клику вне.
- **Профиль (сайдбар)**: Обзор, Основные (Мои объявления, Бронирования, Сообщения, Избранное), Для арендодателя (Продвижение, Финансы), Системные (Настройки, Поддержка). Убраны Аналитика, Админ, дубли доходов.
- **Нижнее меню**: без изменений — 5 пунктов (Главная, Поиск, Добавить, Сообщения, Профиль).
- **Добавить**: ведёт на /create-listing (форма объявления).

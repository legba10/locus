# ТЗ 18 — Архитектура страницы объявления (уровень Airbnb/Cian+)

Только расширение UI, без поломок и смены API/auth.

## Структура страницы (сверху вниз)

1. Галерея (desktop: 1 большое + 4 маленьких; mobile: свайп; счётчик, fullscreen)
2. Заголовок + цена (включая сервис 7%)
3. AI-блок (карточка с иконкой, пункты анализа)
4. Быстрые параметры (гости, спальни, м², этаж)
5. Блок бронирования (правая колонка; даты, гости, итог, кнопка)
6. Описание (или AI-описание если нет текста)
7. Удобства (сетка, «Показать все» при большом списке)
8. Владелец (фото, имя, рейтинг, ответ %; написать, позвонить)
9. Доверие (бейджи: Проверенный объект, AI проверка)
10. Отзывы + AI-анализ отзывов
11. Локация / карта (mock если нет API)
12. Похожие рядом (слайдер)
13. Sticky-панель mobile (цена + забронировать)

## Исправить

- Кривые поля, разная ширина, белые баги в форме бронирования (light/dark одинаково).
- Итоговая цена: X ночей, сервис 7%, итого.

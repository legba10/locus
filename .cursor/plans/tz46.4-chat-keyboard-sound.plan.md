# ТЗ-46.4 — Чат, клавиатура, авто-скролл, звуки

## Цель
Стабилизировать экран сообщений: корректная работа с iOS-клавиатурой, отсутствие zoom на input, предсказуемый авто-скролл, скрытие нижнего меню в чате и стабильные звуки сообщений/уведомлений через единый сервис.

- [completed] Подключить новый модуль в маршруты сообщений без дублирования логики.
- [completed] Реализовать фикс input над клавиатурой (visualViewport) и стабильный layout.
- [completed] Включить авто-скролл при входе/новом сообщении/отправке с условием «не скроллить, если пользователь ушёл вверх».

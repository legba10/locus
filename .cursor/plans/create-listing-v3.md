# Create Listing v3 — план

**Цель:** Самый быстрый флоу размещения на рынке (60–90 сек). Минимальные поля → максимум автоматизации → публикация после модерации.

## Принципы UX
- 1 экран = 1 шаг
- Автосохранение каждые 1–2 сек
- AI-подсказки (Этап 3)
- Минимальный ввод, умные дефолты
- Прогресс-бар

## Статусы (backend уже есть)
- `DRAFT` — черновик
- `PENDING_REVIEW` — на модерации
- `PUBLISHED` — одобрено
- `REJECTED` — отклонено

## Этап 1 (текущий спринт)

| # | Задача | Статус |
|---|--------|--------|
| 1 | Стартовый экран: «Быстро добавить фото» / «Создать с нуля» | done |
| 2 | Фото: сетка мини-карточек [ + ] [ + ] …, drag&drop, reorder, обложка, мин 5 фото | done |
| 3 | Быстрая форма: только Город, Тип, Цена | done |
| 4 | Удобства: категории (кухня, техника, интернет, комфорт) + чипы + «Показать всё» | done |
| 5 | Автосохранение 1–2 сек, прогресс-бар | done |
| 6 | Отправка на модерацию: не вызывать publish после create, текст «Отправить на проверку» | done |

## Этап 2 (модерация и уведомления)
- Уже реализовано: админка, approve/reject, уведомления владельцу/админам.
- При необходимости: таблица `moderation_logs` для истории.

## Этап 3 (AI)
- Определение комнат по фото, автозаполнение по адресу, генерация описания, рекомендация цены.

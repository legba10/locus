# ТЗ 35 — Страница объявления: роли, действия и нижняя панель

## Статус: выполнено

## Реализовано

1. **Конфликты панелей**
   - Глобальное нижнее меню уже скрывается на маршруте объявления (`/listings/[id]`) в `BottomNavGlobal`.
   - На странице объявления используется собственная фиксированная панель действий.

2. **Нижняя панель объявления (mobile)**
   - Высота приведена к `72px`.
   - Фикс внизу (`position: fixed; bottom: 0`), safe-area сохранен.
   - Ролевые режимы:
     - **Гость/обычный пользователь:** `[♡] [Написать] [Забронировать]`
     - **Владелец:** `Редактировать / Календарь / Продвижение / Аналитика`
     - **Админ:** `Одобрить / Отклонить / Редактировать`

3. **Статусы объявления**
   - Добавлены проверки по статусу из данных:
     - `PENDING / ON_REVIEW / MODERATION` → режим модерации.
     - `REJECTED` → отклонено.
     - Иначе (`PUBLISHED` и др.) → обычный режим (без ложного «на модерации»).

4. **Структура и отступы контента**
   - Уменьшен вертикальный разрыв в основном столбце (`space-y-3`).
   - Фото на mobile приведено к `320px`.
   - Контент страницы сохраняет нижний отступ, чтобы не заходить под панель.

5. **Карта**
   - Убрана крупная встроенная карта из потока контента.
   - Кнопка `Показать на карте` добавлена рядом с городом/районом.
   - По нажатию открывается modal с картой, высота карты `50vh`.

6. **Профиль/навигация для связанного UX**
   - В рамках предыдущего шага удалены дубли глобальной навигации:
     - убран side-menu,
     - убран dropdown аватара (аватар ведет в `/profile`).

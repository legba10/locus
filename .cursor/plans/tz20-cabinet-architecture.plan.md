# ТЗ-20 (B) — Архитектура кабинета

Не удалять: сообщения, избранное, объявления, бронирования, AI-метрики, админ. Только реорганизация UI, перенос блоков.

## Уровни

- **Уровень 1:** поиск, избранное, сообщения, профиль
- **Уровень 2:** мои объявления, бронирования, доход, отзывы
- **Уровень 3:** админ (не трогать доступ)

## Новая структура

### Главный экран профиля = дашборд

- Верх: аватар, имя, рейтинг, статус аккаунта
- Под ним 4 карточки: Мои объявления | Бронирования | Сообщения | Доход/аналитика

### Убрать дубли

- Шапка: только сообщения, уведомления, аватар (убрать избранное из шапки)
- Меню авы: профиль, мои объявления, бронирования, выйти (без сообщений/избранного — уже есть)
- Нижнее меню: только Главная, Поиск, Добавить объявление, Сообщения, Профиль. Без дублей (убрать избранное из низа, добавить «Добавить объявление»)

### Этапы

- 20.0 Нижнее меню + шапка (убрать дубли)
- 20.1 Дашборд арендодателя: блок «Мои показатели» (активные объявления, доход, бронирования, рейтинг) — 4 карточки, 4 в ряд ПК / 2 моб
- 20.2 Центр объявлений: карточка с фото, цена, статус, кнопки редактировать/статистика; просмотры, бронирования, доход
- 20.3 AI-аналитика в карточке объявления: спрос, цена рынка, рекомендации
- 20.4 Отзывы и рейтинг в профиле: средний рейтинг, AI-анализ, сильные/слабые стороны
- 20.5 Настройки: имя, фото, пароль, уведомления, тариф; убрать лишние кнопки
- 20.6 Экран дохода: за месяц, за всё время, график

## Ограничения

- НЕ удалять файлы, не менять API, не ломать маршруты
- ТОЛЬКО реорганизация UI, перенос блоков, выравнивание логики

## Выполнено

- Нижнее меню: Главная, Поиск, Добавить объявление, Сообщения, Профиль (без Избранного)
- Шапка: только сообщения, уведомления, аватар (избранное убрано из шапки)
- Профиль = дашборд: аватар, имя, рейтинг, статус; 4 карточки (Мои объявления, Бронирования, Сообщения, Доход); ссылка «Избранное»
- ТЗ-20.1: блок «Мои показатели» для арендодателя (4 карточки: активные объявления, доход, бронирования, рейтинг)
- ТЗ-20.2: в карточке «Мои объявления» — просмотры, бронирования, доход; кнопки Редактировать, Статистика
- ТЗ-20.6: страница /profile/income (за месяц, за всё время, график)
- Сайдбар профиля: Дашборд, Доход, Настройки и др.
- ТЗ-20.3: в карточке объявления (MyListingCard) опциональный блок «AI анализ»: спрос, цена рынка, рекомендации.
- ТЗ-20.4: в дашборде профиля раздел «Отзывы»: средний рейтинг, место под AI-анализ и сильные/слабые стороны.

# ТЗ №10 — Детальная страница объявления (/listings/[id])

## Цель
Сделать страницу объявления чистой, читаемой, без засветов, mobile-first, с логичной структурой и удобной кнопкой бронирования.

## Статус: ✅ выполнено

### Сделано

1. **Галерея фото**
   - `ListingGallery` с `variant="detail"`: height 280px, radius bottom 20, свайп, индикатор точек, стрелки. Одно фото — без свайпа. Плейсхолдер при отсутствии фото.

2. **Цена**
   - Сразу под галереей: «12 000 ₽ / мес», font-size 26px, font-weight 700, `color: var(--text-primary)`.

3. **Кнопка действия**
   - «Забронировать»: на мобиле — sticky внизу экрана; по клику — скролл к блоку бронирования. На десктопе блок бронирования в потоке.

4. **Основная информация**
   - `ListingInfo`: название (var(--text-primary)), подзаголовок «Москва • Таганская» (var(--text-secondary)).

5. **Характеристики**
   - `ListingSpecs`: grid 2 колонки, карточки с текстом («2 комнаты», «45 м²», «3 этаж», «есть кухня», «есть санузел»). Данные собираются из bedrooms, area, floor, totalFloors и amenities.

6. **Описание**
   - Блок «Описание», текст `var(--text-secondary)`.

7. **Владелец**
   - `ListingOwner`: обновлён под тему (var(--text-primary), var(--bg-card), var(--border-main)), имя, статус «онлайн»/«отвечает быстро», кнопки «Написать» и «Профиль».

8. **Карта (mock)**
   - Блок «Район» с текстом района/города.

9. **Тёмная тема**
   - Все тексты и фоны через CSS-переменные, без жёсткого white и лишней opacity.

10. **Компоненты**
    - `ListingGallery.tsx` — добавлен вариант `variant="detail"` (280px, rounded-b-[20], без миниатюр).
    - `ListingInfo.tsx` — заголовок и подзаголовок.
    - `ListingSpecs.tsx` — сетка характеристик.
    - `ListingOwner.tsx` — обновлён под тему и блок «Написать».

11. **Layout**
    - `ListingLayoutTZ10`: галерея → цена → основная инфо → бронирование → характеристики → описание → владелец → район. Sticky-кнопка «Забронировать» внизу на мобиле.

12. **Интеграция**
    - `ListingPageV3` переведён на `ListingLayoutTZ10`: формирование `specItems` из данных объявления, передача owner с lastSeen для «онлайн».

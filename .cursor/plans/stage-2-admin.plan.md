## Задачи

### Dashboard
- Расширить getStats: пользователи, объявления, брони, выручка, **комиссия**, **активные**
- Графики: доход, брони, новые пользователи (неделя/месяц)
- Данные: `GET /api/admin/stats`, агрегаты по датам в backend

### Users
- Таблица (не карточки): имя, email, роль, кол-во объявлений, статус
- Действия: админ/менеджер, бан
- API: существующие роли и бан

### Listings
- Таблица: thumbnail, название, владелец, статус
- Кнопки: открыть, удалить, бан
- API: модерация/удаление

### Unit-экономика
- Блок: total revenue, комиссия, средний чек, кол-во бронирований
- Данные из getStats/economy

### AI-помощник
- `frontend/app/admin/ai/page.tsx`
- Ответы: сколько бронирований, сколько денег, популярные объявления
- Backend AI/статистика, без выдуманных данных

# ТЗ-46.5 — Глобальная система звуков

## Цель
Собрать единую архитектуру звуков сообщений/уведомлений/входа через singleton sound engine с инициализацией один раз на приложение, безопасным проигрыванием, iOS/Safari unlock и защитой от дублей.

- [completed] Перевести провайдер приложения на единичную инициализацию sound engine.
- [completed] Обновить auth-потоки (login/refresh/session enter) на единый login sound без дублей рендера.

- [completed] Добавить `user.settings.sound` флаг по умолчанию (`true`) в контракт/модель пользователя.

# Stagewise — фронтенд-агент в проекте LOCUS

[Stagewise](https://stagewise.io/docs) — агент для продакшен-кодбейзов с контекстом из браузера.

## Как использовать

1. **Запусти приложение в dev-режиме** (в одном терминале):
   ```bash
   npm run dev
   ```

2. **В другом терминале перейди в корень фронтенда** (здесь лежит `package.json`):
   ```bash
   cd frontend
   ```

3. **Запусти Stagewise**:
   ```bash
   npm run stagewise
   ```
   или напрямую:
   ```bash
   npx stagewise@latest
   ```

4. Пройди короткую настройку в CLI (аккаунт stagewise) и работай: говори агенту, что менять, кликай по элементам в браузере — агент получит контекст и внесёт правки.

## Важно

- Stagewise должен запускаться **из папки фронтенда** (`frontend/`), где находится `package.json` Next.js-приложения.
- Приложение должно быть запущено в режиме разработки (`npm run dev`).
